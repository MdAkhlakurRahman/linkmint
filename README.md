# LinkMint

A production-ready URL shortening and analytics platform built with Java 21, Spring Boot, PostgreSQL, Redis, and Docker.

## 🚀 Tech Stack

- Java 21
- Spring Boot
- PostgreSQL
- Redis
- Docker
- Maven

## 📌 Project Status

🚧 Under active development.

## 📅 Development Workflow
